L, R    Rotate page left (clockwise) or right (counter-clockwise).

/       Start searching forward.
?       Start searching backward.
n       Continue searching forward.
N       Continue searching backward.

+, -	Zoom in or out.
W,H		fit to width, heigth
Z		autofit (fit to height if the page is vertical, to width if the page is horizontal)
w		resize window to fit the page
f		fullscreen

[n]g
[n]ENTER    go to page n (will go to the first page with no number)
G           go to last page
